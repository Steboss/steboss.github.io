# Countdown Exercise Answer

{% highlight python %}
for  i in range(10,0,-1):
    print(i)
{% endhighlight %}

If you wish a prettier countdown:

{% highlight python %}
for i in range(10,0,-1):
    print(i)
print("Happy New Yearrrr!")
{% endhighlight %}

and so on...here we can play a lot with the main theme :)
